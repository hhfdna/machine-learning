## 项目说明
`最近在学习machine learning in action，书中有很多代码实例，为了方便理解，我使用python3.6环境对代码进行实际运行和调试。
而原书中使用的python语法是基于2.7的，python的语法有很多变化。例如：对于列表或者字典对象的迭代`

```python
classCount.iteritems()
```

这个api在python3.6中已经没有了，变为
```python
classCount.items()
```
`所以对书中部分代码进行了修改。`

另外增加了一些自己写的测试程序。
